* <!DOCTYPE html> -- DOCTYPE is an "information" to the browser about what document type to expect.
* <html></html> -- The html element represents the root of an HTML document.
* <head></head> -- The head element represents a collection of metadata for the document.
	<meta>, <style>, <link>, <title>, <script>, <base>
* <body></body> -- The body element represents the content for the document.

* what is attributes?
 Ans:Attributes provide additional information about an element. All HTML elements can have attributes. Attributes are always specified in the start tag.
	Attributes Name = "value"
